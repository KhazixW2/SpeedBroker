## 策略选择矩阵

### 根据市场类型选择

```
趋势市场（强上涨/下跌）
├─ 短期: 双均线 → MACD
└─ 长期: 三均线 → 海龟交易

震荡市场（横盘整理）
├─ 短期: 布林带 → RSI → KDJ
└─ 长期: 均值回归

牛市（持续上涨）
├─ 短期: MACD
└─ 长期: 动量策略

不确定市场
└─ 中期: 组合策略
```

### 根据投资风格选择

```
保守型（稳健收益）
├─ 三均线策略（长期）
├─ 海龟交易策略（长期）
└─ 均值回归（中期）

平衡型（风险收益平衡）
├─ 双均线策略（短中期）
├─ MACD策略（短期）
└─ 组合策略（中期）

激进型（高风险高收益）
├─ KDJ策略（短期）
├─ RSI策略（短期）
└─ 动量策略（长期牛市）
```

## 实战建议

### 新手推荐路径

1. **第一阶段**: 双均线策略
   - 理解基本概念
   - 学习回测分析
   - 建立交易纪律

2. **第二阶段**: 三均线策略
   - 提升风险控制
   - 减少交易频率
   - 适应长期持仓

3. **第三阶段**: MACD或组合策略
   - 理解多指标配合
   - 优化参数设置
   - 提高策略灵活性

### 进阶用户建议

1. **多策略组合**
   - 30% 三均线（稳定收益）
   - 40% MACD或双均线（主要收益）
   - 30% 均值回归或布林带（补充收益）

2. **动态调整**
   - 牛市: 增加动量策略权重
   - 震荡市: 增加均值回归和布林带权重
   - 熊市: 降低仓位，使用三均线

### 常见错误

❌ **不要做的事**:
1. 频繁切换策略
2. 过度优化参数（过拟合）
3. 忽视交易成本
4. 不设止损
5. 追求完美的策略

✅ **应该做的事**:
1. 选择适合自己的策略并坚持
2. 定期回测和评估
3. 合理控制仓位
4. 严格执行纪律
5. 持续学习改进

## 使用流程

### 1. 策略测试流程

```bash
# 步骤1: 选择策略并配置参数
编辑 config.ini
strategy_name = MACD  # 选择策略

# 步骤2: 运行回测
python main.py

# 步骤3: 查看结果
查看 output/ 目录下的图表和CSV文件
```

### 风险提示

⚠️ **重要提醒**:
1. 过去的表现不代表未来收益
2. 所有策略都有其失效的可能
3. 务必进行充分的回测验证
4. 实盘前请小资金测试
5. 永远不要投入超过承受能力的资金

### 最终建议

💡 **成功的关键**:
- **新手**: 从三均线开始，建立信心
